Sistema de Partículas

⏱️ 25 minutos • Programación Orientada a Objetos

🎯 Objetivos

  • Crear clase Star con propiedades y métodos
  • Generar 100 estrellas en posiciones aleatorias
  • Animar parpadeo con función sin()

🧠 Patrón de Diseño: Object Pool

Creamos objetos una vez y los reutilizamos, en lugar de crear/destruir constantemente. Esto mejora el rendimiento.

Paso 1: Crea la clase Star con constructor, update() y display()

Paso 2: En setup(), llena el array stars con 100 instancias

Paso 3: En draw(), itera el array llamando a ambos métodos

Paso 4: Agrega efecto de "trail" con background(r,g,b,alpha)

    <script>
        // ==========================================
        // BLOQUE 2: Sistema de Partículas - Estrellas
        // ==========================================
        
        // 1. Variables globales
        let stars = [];      // Array para guardar estrellas
        let starColor;       // Color compartido

        function setup() {
            let canvas = createCanvas(windowWidth, windowHeight);
            canvas.parent('p5-container');
            
            // 2. Inicializar color (primary de Tailwind)
            starColor = color(222, 183, 255);
            
            // 3. Crear 100 estrellas
            // Usamos 'new' para instanciar la clase Star
            for (let i = 0; i < 100; i++) {
                stars.push(new Star());
            }
            
            console.log('✅ Sistema de partículas:', stars.length, 'estrellas');
        }

        function draw() {
            // 4. Fondo semi-transparente para efecto "trail"
            // El cuarto valor (30) es la opacidad (0-255)
            background(23, 13, 44, 30);
            
            // 5. Iterar sobre TODAS las estrellas
            // 'of' nos da el objeto directamente, no el índice
            for (let star of stars) {
                star.update();    // Actualizar estado
                star.display();   // Dibujar en pantalla
            }
            
            // 6. Cursor personalizado (mejorado)
            noCursor();                    // Ocultar cursor default
            fill(starColor);
            noStroke();
            circle(mouseX, mouseY, 20);    // Núcleo
            
            // Anillo exterior
            noFill();
            stroke(starColor);
            strokeWeight(2);
            circle(mouseX, mouseY, 40);
        }

        function windowResized() {
            resizeCanvas(windowWidth, windowHeight);
        }

        // ==========================================
        // CLASE STAR: Define comportamiento de cada partícula
        // ==========================================
        class Star {
            constructor() {
                // POSICIÓN: Aleatoria en todo el canvas
                this.x = random(width);
                this.y = random(height);
                
                // TAMAÑO: Entre 0.5 y 3 píxeles
                this.size = random(0.5, 3);
                
                // PARPADEO: Valores para animación senoidal
                this.baseAlpha = random(100, 255);  // Opacidad base
                this.alpha = this.baseAlpha;        // Opacidad actual
                this.twinkleSpeed = random(0.02, 0.08);  // Velocidad
                this.twinkleOffset = random(TWO_PI);     // Fase inicial
                
                // COLOR: Referencia al color global
                this.col = starColor;
            }

            update() {
                // Efecto de parpadeo usando función seno
                // sin() oscila entre -1 y 1
                this.twinkleOffset += this.twinkleSpeed;
                this.alpha = this.baseAlpha + sin(this.twinkleOffset) * 50;
                
                // Limitar alpha entre 50 y 255 (nunca invisible ni saturada)
                this.alpha = constrain(this.alpha, 50, 255);
            }

            display() {
                noStroke();
                
                // NÚCLEO BRILLANTE
                fill(red(this.col), green(this.col), blue(this.col), this.alpha);
                circle(this.x, this.y, this.size);
                
                // GLOW: Solo para estrellas grandes (> 2px)
                if (this.size > 2) {
                    // Círculo más grande, más transparente
                    fill(red(this.col), green(this.col), blue(this.col), 
                         this.alpha * 0.3);
                    circle(this.x, this.y, this.size * 3);
                }
            }
        }
    </script>

📊 Fórmula del Parpadeo 

alpha = base + sin(ángulo) × amplitud

Ejemplo:
200 + sin(3.14) × 50 = 150
200 + sin(0) × 50 = 200

Interacción con Mouse

⏱️ 20 minutos • Eventos y estados dinámicos

🎯 Objetivos

  • Detectar clicks con mousePressed()
  • Crear estrellas "nuevas" con velocidad y vida limitada
  • Actualizar contador en el DOM (HTML ↔ p5.js)

⚠️ Cambios en la Clase Star

Modificamos el constructor para aceptar parámetros. Las estrellas ahora pueden ser:

  • Estáticas: Creadas al inicio, viven para siempre
  • Dinámicas: Creadas por click, se mueven y mueren

Paso 1: Modificar constructor: new Star(x, y, isNew)

Paso 2: Agregar propiedades: vx, vy, life, isNew

Paso 3: Crear función mousePressed()

Paso 4: Crear updateStarCount() para modificar el DOM

Modificaciones al script anterior 
    <script>
        // ==========================================
        // BLOQUE 3: Interacción con Mouse
        // ==========================================
        
        let stars = [];
        let starColor;
        let currentColor;  // Para el cursor

        function setup() {
            let canvas = createCanvas(windowWidth, windowHeight);
            canvas.parent('p5-container');
            
            starColor = color(222, 183, 255);
            currentColor = starColor;
            
            // Estrellas estáticas iniciales
            for (let i = 0; i < 100; i++) {
                stars.push(new Star());  // Sin parámetros = estática
            }
        }

        function draw() {
            background(23, 13, 44, 30);
            
            // Iterar y dibujar
            for (let star of stars) {
                star.update();
                star.display();
            }
            
            // Cursor (sin cambios)
            noCursor();
            noFill();
            stroke(currentColor);
            strokeWeight(2);
            circle(mouseX, mouseY, 40);
            fill(currentColor);
            noStroke();
            circle(mouseX, mouseY, 20);
        }

        // ==========================================
        // NUEVO: Evento de click
        // ==========================================
        function mousePressed() {
            console.log('🖱️ Click en:', mouseX, mouseY);
            
            // Crear 5 estrellas "nuevas" en posición del mouse
            for (let i = 0; i < 5; i++) {
                // Parámetros: x, y, isNew=true
                stars.push(new Star(mouseX, mouseY, true));
            }
            
            // Actualizar contador en HTML
            updateStarCount();
        }

        // ==========================================
        // NUEVO: Comunicación con DOM
        // ==========================================
        function updateStarCount() {
            // Buscar elemento por ID
            let countElement = document.getElementById('star-count');
            
            // Si existe, actualizar su texto
            if (countElement) {
                countElement.textContent = stars.length;
            }
        }

        function windowResized() {
            resizeCanvas(windowWidth, windowHeight);
        }

        // ==========================================
        // CLASE STAR MODIFICADA
        // ==========================================
        class Star {
            // Constructor ahora acepta parámetros opcionales
            constructor(x, y, isNew = false) {
                
                // Si es NUEVA (creada por click)
                if (isNew) {
                    // Posición cercana al mouse (con dispersión)
                    this.x = x + random(-20, 20);
                    this.y = y + random(-20, 20);
                    
                    // Velocidad aleatoria en todas direcciones
                    this.vx = random(-3, 3);
                    this.vy = random(-3, 3);
                    
                    // Vida limitada (se desvanece)
                    this.life = 255;
                    this.isNew = true;
                    
                } else {
                    // Si es ESTÁTICA (creada al inicio)
                    this.x = random(width);
                    this.y = random(height);
                    this.vx = 0;
                    this.vy = 0;
                    this.life = -1;  // -1 = infinito
                    this.isNew = false;
                }
                
                // Propiedades comunes (sin cambios)
                this.size = random(0.5, 3);
                this.baseAlpha = random(100, 255);
                this.alpha = this.baseAlpha;
                this.twinkleSpeed = random(0.02, 0.08);
                this.twinkleOffset = random(TWO_PI);
                this.col = starColor;
            }

            update() {
                // COMPORTAMIENTO DIFERENTE según tipo
                
                if (this.isNew) {
                    // 1. Moverse según velocidad
                    this.x += this.vx;
                    this.y += this.vy;
                    
                    // 2. Reducir vida
                    this.life -= 3;  // Se desvanece rápido
                    
                    // 3. Opacidad = vida actual
                    this.alpha = this.life;
                    
                } else {
                    // Estática: solo parpadear (código anterior)
                    this.twinkleOffset += this.twinkleSpeed;
                    this.alpha = this.baseAlpha + sin(this.twinkleOffset) * 50;
                    this.alpha = constrain(this.alpha, 50, 255);
                }
            }

            display() {
                // No dibujar si está "muerta"
                if (this.alpha <= 0) return;
                
                noStroke();
                fill(red(this.col), green(this.col), blue(this.col), this.alpha);
                circle(this.x, this.y, this.size);
                
                if (this.size > 2 || this.isNew) {
                    fill(red(this.col), green(this.col), blue(this.col), this.alpha * 0.3);
                    circle(this.x, this.y, this.size * 3);
                }
            }
            
            // NUEVO: Método para saber si debe eliminarse
            isDead() {
                return this.isNew && this.life <= 0;
            }
        }
    </script>

Stats en Tiempo Real + Optimizaciones

⏱️ 20 minutos • Performance y pulido

🎯 Objetivos

  • Mostrar FPS (frames por segundo)
  • Mostrar posición del mouse en tiempo real
  • Optimizar: no actualizar DOM cada frame

⚡ Por qué optimizar?

draw() se ejecuta 60 veces por segundo. Modificar el DOM (element.textContent) es lento. Si actualizamos stats cada frame, bajamos los FPS.

Solución: Actualizar cada N frames usando el operador módulo %

Paso 1: Actualizar HTML de stats para tener IDs funcionales

Paso 2: Crear función updateStats() unificada

Paso 3: Usar frameCount % 10 === 0 para limitar actualizaciones

Paso 4: Agregar fricción a estrellas nuevas para movimiento más natural

Script completo - Versión Final 
    <script>
        // ==========================================
        // BLOQUE 5: Versión Final Optimizada
        // ==========================================
        
        let stars = [];
        let starColor;
        let currentColor;
        
        // OPTIMIZACIÓN: Contador de frames para no saturar DOM
        let frameCounter = 0;

        function setup() {
            let canvas = createCanvas(windowWidth, windowHeight);
            canvas.parent('p5-container');
            
            starColor = color(222, 183, 255);
            currentColor = starColor;
            
            for (let i = 0; i < 100; i++) {
                stars.push(new Star());
            }
            
            // Inicializar contador de estrellas
            updateStarCount();
        }

        function draw() {
            background(23, 13, 44, 30);
            
            // Actualizar estrellas (iteración inversa para eliminar)
            for (let i = stars.length - 1; i >= 0; i--) {
                stars[i].update();
                stars[i].display();
                
                if (stars[i].isDead()) {
                    stars.splice(i, 1);
                }
            }
            
            // Dibujar cursor
            noCursor();
            noFill();
            stroke(currentColor);
            strokeWeight(2);
            circle(mouseX, mouseY, 40);
            fill(currentColor);
            noStroke();
            circle(mouseX, mouseY, 20);
            
            // ==========================================
            // OPTIMIZACIÓN: Actualizar stats cada 10 frames
            // ==========================================
            frameCounter++;
            if (frameCounter % 10 === 0) {
                updateStats();
                updateStarCount();
            }
        }

        function mousePressed() {
            for (let i = 0; i < 5; i++) {
                stars.push(new Star(mouseX, mouseY, true));
            }
        }

        function changeColor(hexColor) {
            let newColor = color(hexColor);
            currentColor = newColor;
            starColor = newColor;
            
            for (let star of stars) {
                star.col = newColor;
            }
        }

        // ==========================================
        // NUEVO: Función unificada de stats
        // ==========================================
        function updateStats() {
            // FPS (frames por segundo)
            let fpsElement = document.getElementById('fps-count');
            if (fpsElement) {
                // frameRate() devuelve un float, lo redondeamos
                fpsElement.textContent = Math.floor(frameRate());
            }
            
            // Posición del mouse
            let posElement = document.getElementById('mouse-pos');
            if (posElement) {
                posElement.textContent = mouseX + ',' + mouseY;
            }
        }

        function updateStarCount() {
            let countElement = document.getElementById('star-count');
            if (countElement) {
                countElement.textContent = stars.length;
            }
        }

        function windowResized() {
            resizeCanvas(windowWidth, windowHeight);
        }

        class Star {
            constructor(x, y, isNew = false) {
                if (isNew) {
                    this.x = x + random(-20, 20);
                    this.y = y + random(-20, 20);
                    this.vx = random(-3, 3);
                    this.vy = random(-3, 3);
                    this.life = 255;
                    this.isNew = true;
                } else {
                    this.x = random(width);
                    this.y = random(height);
                    this.vx = 0;
                    this.vy = 0;
                    this.life = -1;
                    this.isNew = false;
                }
                
                this.size = random(0.5, 3);
                this.baseAlpha = random(100, 255);
                this.alpha = this.baseAlpha;
                this.twinkleSpeed = random(0.02, 0.08);
                this.twinkleOffset = random(TWO_PI);
                this.col = starColor;
            }

            update() {
                if (this.isNew) {
                    this.x += this.vx;
                    this.y += this.vy;
                    this.life -= 3;
                    this.alpha = this.life;
                    
                    // ==========================================
                    // MEJORA: Fricción para movimiento más natural
                    // ==========================================
                    // Multiplicar velocidad por 0.98 = reducir 2% cada frame
                    this.vx *= 0.98;
                    this.vy *= 0.98;
                    
                } else {
                    this.twinkleOffset += this.twinkleSpeed;
                    this.alpha = this.baseAlpha + sin(this.twinkleOffset) * 50;
                    this.alpha = constrain(this.alpha, 50, 255);
                }
            }

            display() {
                if (this.alpha <= 0) return;
                
                noStroke();
                fill(red(this.col), green(this.col), blue(this.col), this.alpha);
                circle(this.x, this.y, this.size);
                
                if (this.size > 2 || this.isNew) {
                    fill(red(this.col), green(this.col), blue(this.col), this.alpha * 0.3);
                    circle(this.x, this.y, this.size * 3);
                }
            }
            
            isDead() {
                return this.isNew && this.life <= 0;
            }
        }
    </script>

5 Checklist de Optimización

✅ Hacer

  • Usar canvas.parent()
  • Iterar al revés al eliminar
  • Limitar updates de DOM
  • Usar clear() si es necesario
  • Reciclar objetos cuando sea posible

❌ Evitar

  • Crear canvas sin parent
  • Modificar DOM cada frame
  • Dejar objetos sin eliminar
  • Usar background() si quieres transparencia
  • Arrays que crecen infinitamente

🛠️ Debug

  • console.log() para variables
  • frameRate() para performance
  • print() en p5.js
  • Inspector del navegador (F12)
  • Consola de p5.js (activar en settings)
